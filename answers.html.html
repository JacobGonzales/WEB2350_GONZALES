What is the difference between TCP/IP and HTTP?

HTTP is used for surfing the Internet. HTTP transfers information via the link which TCP provides using an IP address.

Define the term: URL:

It is basically an address in the World Wide Web. (i.e. www.google.com)

What is the Client/Server Model?

A client server model can be described as a user that is on a computer that requests information. The request goes to a server then the information that was requested was given back. Simply put it is a client and server exchange.

What two things does the browser do?

  1. Request information on the internet via URL.
  2. Display information from the internet via URL.

Terms:

  • HTML - Hypertext Markup Language that is used to create documents.
  • CSS - Cascading Style Sheets is a style sheet language used for visual design.
  • JavaScript - Object Orienting computer programming language to make interactive effects on web browsers.
